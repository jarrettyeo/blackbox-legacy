# Blackbox
#### Dropbox for Blackboard

___

## **How to Use**

Windows:

```
$ pip install selenium
$ cd path/to/working/directory
$ python blackbox.py
```

Mac:

```
$ sudo pip3 install selenium
$ cd path/to/working/directory
$ python3 blackbox.py
```
